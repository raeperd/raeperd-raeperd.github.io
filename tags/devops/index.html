About Vagrant


  • devops

Motivation 최근에 CentOS8 뿐만 아니라 CentOS6 (기존 환경) 에서도 동작가능한 C++ 프로그램을 개발할 필요가 있었습니다. 기존 환경의 경우 설치스크립트를 이용해 초기화를 진행하는데 CentOS8 의 경우 아직 설치 스크립트가 없기 때문에 처음부터 설치 스크립트를 작성하면서 환경을 구성해야 합니다. 개발과 빌드를 하면서 OS의 설치 스크립트를 같이 관리하는 것은 여러므로 귀찮은 일이라 처음에는 Docker를 적용하려 했습니다. 가능하다면 이 방법이 최선이겠지만, Virtual Machine 에 의존성을 가지고 있는 실행프로그램이기 때문에 대안을 찾아야만 했습니다. 이 과정에서 Vagrant 를 알게 되었고 현재는 Vagrant 를 활용해 개발 환경을 구성했습니다....
