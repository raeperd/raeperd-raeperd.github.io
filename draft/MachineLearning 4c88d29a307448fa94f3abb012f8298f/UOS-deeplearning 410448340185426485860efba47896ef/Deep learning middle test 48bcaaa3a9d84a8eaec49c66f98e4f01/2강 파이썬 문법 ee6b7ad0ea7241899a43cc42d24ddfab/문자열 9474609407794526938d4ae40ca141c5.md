# 문자열

### 특수기호 출력

```python
food = 'Python\'s favorite food is perl'
print(food)

say = "\"Python is very easy.\" he says."
print(say)
```

- `\`로 할 수 있다.


### 문자열 포맷팅

```python
print("I eat %d apples." % 3)
print("I eat %s apples." % "five")

number = 10
day = "three"
print("I ate %d apples. so I was sick for %s days." % (number, day))

print("I eat {0} apples".format(3))
print("I eat {0} apples".format("five"))

number = 3
print("I eat {0} apples".format(number))

number = 10
day = "three"
print("I ate {0} apples. so I was sick for {1} days.".format(number, day))
```

- 기타 문자열 포맷팅은 C언어의 `printf`와 유사함

### 문자열 관련 함수들

### count()

```python
# 특정문자 갯수 세기
a = "hobby"
print(a.count('b'))
```

- 문장에서 특정 문자 및 문자열의 갯수 세기

### find() & index()

```python
# 특정문자 위치찾기(fnd와 index)
a = "Python is best choice"

print(a.find('b'))   # 처음 나온 위치(0부터 인덱싱)
print(a.find('k'))   # 존재하지 않으면 -1

#  find와 다른점은 존재하지 않으면 에러발생
print(a.index('b'))   # 처음 나온 위치(0부터 인덱싱)
print(a.index('k'))   # 존재하지 않으면 에러
```

### join()

```python
# 문자열 삽입(join)
a= ","
b = a.join('abcd')
print(b)
```

- 구분자를 끼워넣을 수 있다.

### upper() & lower()

```python
# 대소문자 바꾸기(upper, lower)
a = "hi"
b = a.upper()
print(b)
```

### strip() lstrip() rstrip()

```python
# 공백지우기(strip, lstrip, rstrip)
a = " hi "
b = a.lstrip()
print(b)
```

- `strip` 은 양쪽 모두 지워준다.

### replace()

```python
#문자열 바꾸기(replace)
a = "  인생은 너무 짧다"
a = a.strip()
print(a)
b = a.replace("인생", "life")
print(b)
```

### split()

```python
# 문자열 나누기(split)
a = "Life is too short"
b = a.split()  # 기본은 공백(스페이스, 탭, 엔터등)을 기준으로 분리
print(b)

a = "a:b:c;d"
b = a.split(":")
print(b)
```

- 리스트로 반환해 준다.
