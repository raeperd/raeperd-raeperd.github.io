# tensor

example 

```python
import numpy as np

x1 = np.array([2,3,4])
x2 = np.array([[2,3,4],
               [5,6,7]])
x3 = np.array([x2,x2])

print(x1.shape)
print(x2.shape)
print(x3.shape)
```

- shape는 우에서 좌로 차원을 늘려간다.
    - 가장 왼쪽이 높은 차원임
- N-d tensor는 (N-1)-D tensor 를 원소로 가진다.
- np.array의 원소의 차원이 다르면 list로 바뀐다.
