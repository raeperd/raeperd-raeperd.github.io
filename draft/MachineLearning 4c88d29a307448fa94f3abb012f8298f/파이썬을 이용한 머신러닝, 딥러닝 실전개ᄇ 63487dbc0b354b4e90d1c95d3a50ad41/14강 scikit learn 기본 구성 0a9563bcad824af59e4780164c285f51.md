    [0,0],

## How to use `metrics.accuracy_score`

```python
from sklearn import svm, metrics

datas = [ [0,0], [0,1], [1,0], [1,1] ]
labels = [0, 1, 1, 0]
examples = [[0,0], [1,0]]
examples_label = [0, 1]             # answer

clf = svm.SVC()
clf.fit([
    [0,0],
    [0,1],
    [1,0],
    [1,1]
], [0, 1, 1, 0])
results = clf.predict([
    [0,0],
    [0,1]
])

score = metrics.accuracy_score(examples_label, results)
print("score rate : ", score)
```
